# README

## Installation

Para instalar el proyecto siga los siguientes pasos:

1. Clonar el repositorio

```bash
git clone https://github.com/xOverlap/visor-proyect.git
```

2. Entrar a la carpeta del proyecto

```bash
cd visor-proyect
```

3. Instalar las dependencias

```bash
npm install
```
